
##### 1. Copy the Project

```sh
git clone https://github.com/shashanksangawar/StudyWings
```


##### 2. Install Dependencies on Frontend

```sh
cd StudyWings/Frontend
npm install
```

##### 3. Run Frontend

```sh
npm run dev
```


##### 4. Install Dependencies on Backend

```sh
cd ../Backend
npm install
```


##### 5. Run Backend

```sh
npm test
```
